Modelul cutiei

În modelul cutiei, orice element este un container dreptunghiular, indiferent de modul în care este prezentat vizual pe pagină. Un element are o zonă de conținut și margini opționale, umplutură și o bordură (chenar).

Intre top si bottom si intre left si right, de la exterior spre interior, avem :

  • Margin

    Marginea (margin) – adaugă o margine înafara elementului, de la marginea exterioară (bordură) la elementele adiacente, separându-le astfel pe pagină. Poate avea valori negative. Fundalul elementului nu se extinde niciodată până la margine.

  • Border

    Bordura (border) – chenarul elementului. În mod implicit, lățimea sa este zero. Dacă nu este specificată nicio culoare a bordurei, aceasta capătă culoarea conținutului principal, cum ar fi textul.

  • Padding

    Umplutura (padding) – distanța dintre conținutul unui element și bordura acestuia. E folosit pentru efecte decorative și nu poate avea valori negative.

  • Content (cu prorietatile width si height)

    Zona de conținut (content) – conținutul unui element, cum ar fi un text, o imagine, un videoclip sau elemente suprapuse. În mod implicit, înălțimea zonei de conținut este determinată de conținut și lățimea este determinată de tipul elementului (bloc sau inline).
Exemplu

This is the content of an element, such as text, image, video, or nested elements. By default, the height of the content area is determined by the content, and the width is determined by the element type.

HTML

<p class="text">This is the content of an element, such as text, image, video, or nested elements. By default, the height of the content area is determined by the content, and the width is determined by the element type.</p>

CSS

body {
font-family: sans-serif;
}

.text {
border: 20px dashed #2196F3;
padding: 15px;
margin: 20px;
}

Interesant...!!!

Unele elemente din foaia de stil a browserului dețin umplutură și margine ce au dimensiuni setate în mod implicit. De exemplu, listele, paragrafele, titlurile etc. Acest lucru a fost necesar pentru a ajuta conținutul textual al paginii să arate uniform, fără un stil suplimentar.

  • Proprietățile width și height

    Aceste proprietăți vă permit să specificați în mod explicit lățimea și înălțimea totală a unui element sau doar dimensiunea zonei de conținut (content) a acestuia, în funcție de tipul modelului cutiei.

    Câteva sfaturi pentru utilizarea acestor proprietăți:

    • Nu setați elementelor o înălțime fixă, ci lăsați-o să fie determinată de conținut. O înălțime codificată într-un mod strict duce la o problemă de depășire a spațiului atunci când există mai mult conținut decât e capabil elementul să cuprindă.

    • Majoritatea elementelor primesc doar o lățime. Specificarea simultană a width și height merită doar pentru elementele decorative cu o dimensiune fixă, de exemplu pentru iconițe.

    • Lățimea poate fi specificată nu numai în px, em sau rem, ci și în procente. În acest caz, lățimea blocului va fi calculată în procente din lățimea părintelui. Veți afla mai multe despre asta atunci când vom construi grile.
    Exemplu
    HTML

    <div class="box"></div>

    CSS

    .box {
    width: 150px;
    height: 150px;

    border: 5px solid #f44336;
    padding: 30px;
    margin: 40px;
    background-color: #ffeb3b;
    }

    Atentie...!!!

    Lățimea și înălțimea finală a blocului div.box va fi de 220px, nu de 150px, așa cum se specifică în proprietățile width și height. De fapt, există mai multe tipuri de modele de bloc care diferă prin formula de calcul a dimensiunii unui element.

  • Proprietatea box-sizing

    Setează tipul modelului de cutie al elementului – formula pentru calcularea dimensiunilor acestuia.

    box-sizing: content-box | border-box | inherit

      • content-box – proprietățile width și height setează lățimea și înălțimea zonei de conținut și nu includ dimensiunile umpluturii (padding), a bordurii (border) și marginii (margin).
      • border-box – proprietățile width și height setează dimensiunea finală a elementului și includ valorile umpluturii (padding) și a bordurii (border), dar nu și a marginii (margin).
      • inherit – moștenește valoarea modelului cutie al strămoșului.

    • Valoarea content-box

      Acest tip de model de cutie este utilizat de browsere în mod implicit. Calcularea dimensiunii finale a unui element este mai dificilă, deoarece proprietățile width și height nu setează dimensiunea totală a elementului, ci doar înălțimea și lățimea zonei sale de conținut (content).

      Formulele pentru calcularea lățimii și înălțimii finale a blocului, atunci când valoarea este content-box.

      Exemplu
      HTML

      <div class="content-box"></div>

      CSS

      /* Lățimea finală a blocului = 150px + 30px + 30px + 5px + 5px */
      .content-box {
      width: 150px;
      height: 150px;
      border: 5px solid #f44336;
      padding: 30px;
      margin: 100px;
      background-color: #ffeb3b;
      }

      În devTools, în fila Computed, este afișată geometria acestui element.

      Lațimea = width + padding-left + padding-right + border-left + border-right

      Înălțimea = height + padding-top + padding-bottom + border-top + border-bottom

      Asa ne-a rezultat mai sus inaltimea si latimea finala a blocului div .box de 220 px

    • Valoarea border-box

      În acest model, proprietățile width și height stabilesc dimensiunile finale ale elementului și nu a zonei de conținut. Conținutul "se micșorează" automat, astfel încât padding și border să se potrivească în interiorul containerului.

      Formulele pentru calcularea lățimii și înălțimii finale a cutiei atunci când valoarea este border-box.

      Lățimea = width

      Înălțimea = height

      Prin setarea proprietății box-sizing: border-box, schimbăm tipul modelului de cutie al elementului curent.

      Exemplu
      HTML

      <div class="border-box"></div>

      CSS

      .border-box {
      box-sizing: border-box;
      width: 150px;
      height: 150px;
      border: 5px solid #f44336;
      padding: 30px;
      margin: 100px;
      background-color: #ffeb3b;
      }

      În devTools, în fila Computed, este afișată geometria acestui element.

    • Valoarea globală border-box

      Modelul standard al cutiei în proiectele moderne este border-box. Pentru a nu seta proprietatea box-sizing pentru fiecare element în parte chiar la începutul fișierului de stil, se folosește următorul cod CSS.

      *,
      *::before,
      *::after {
      box-sizing: border-box;
      }

      /* Restul codului */

      Folosind selectorul universal *, valoarea border-box este setată pentru toate elementele și pseudo-elementele (mai multe despre asta mai târziu) din document.

  • Normalizarea stilurilor

    Ne dorim ca indiferent de browser, elementele HTML să arate la fel. Din păcate, nu este cazul, iar aceleași elemente pot arăta ușor diferit în browsere diferite. Atunci când un browser afișează o pagină web, se aplică mai întâi o foaie de stil implicită (foaia de stil agent de utilizator), iar valorile proprietăților acesteia pot fi diferite de foile de stil ale altor browsere. Acest lucru cauzează diferențe ușoare în aspectul unor elemente.

    Normalizarea stilului este o încercare de a rezolva această problemă prin includerea unui alt fișier de stil care remediază eventualele discrepanțe între foile de stil ale browserului. Adică este o tehnică de îmbunătățire a compatibilității cross-browser a unei pagini web – aceeași afișare și funcționare a site-ului în browsere diferite.

    Tot ce trebuie să faceți este să includeți încă un fișier de stil înaintea tuturor stilurilor.

    <head>
    <!-- Mai întâi normalizarea -->
    <link
    rel="stylesheet"
    href="https://cdnjs.cloudflare.com/ajax/libs/modern-normalize/1.0.0/modern-normalize.min.css"
    />
    <!-- Apoi stilurile -->
    <link rel="stylesheet" href="link la foaia de stil" />
    </head>

    Util...!!!

    În normalizarea stilurilor, modern-normalize, valoarea globală border-box este setată în mod implicit, de aceea, nu trebuie neapărat să fie menționată printre stiluri.

Geometria unui element

Geometria unui element cuprinde 4 laturi: sus (top), dreapta (right), jos (bottom) și stânga (left). În această ordine, în sensul acelor de ceasornic, începând de sus, sunt setate valorile proprietăților compuse padding, margin și border.

  • Proprietatea padding

    PADDING: TOP RIGHT BOTTOM LEFT

    O proprietate compusă care controlează umplutura dintre conținut și bordură, pentru toate laturile unui element. Rezultatul lucrării depinde de numărul de valori al proprietății.

    /* Aceleași umpluturi de 20px pentru toate părțile */
    padding: 20px;

    /* Umpluturile de sus și de jos 10px, stânga și dreapta 20px */
    padding: 10px 20px;

    /* Umplutura de sus 10px, stânga și dreapta 20px, de jos 5px */
    padding: 10px 20px 5px;

    /* Umplutura de sus 10px, dreapta 15px, de jos 20px, stânga 25px */
    padding: 10px 15px 20px 25px;

    De asemenea, există proprietăți pentru setarea separată a umpluturii fiecărei părți.

    padding-top: 20px;
    padding-right: 10px;
    padding-bottom: 15px;
    padding-left: 30px;

    Câmpurile (paddings) sunt folosite pentru efecte decorative. De exemplu, pentru a crea un spațiu liber între bordura unui element și conținutul acestuia, ca în exemplul de mai sus. Dacă nu setați umplutura, atunci textul va arăta inestetic față de bordura paragrafului.

  • Proprietatea margin

    O proprietate compusă care controlează marginea externă a bordurii pentru toate laturile unui element. Rezultatul depinde de numărul de valori al proprietății.

    /* Aceleași margini de 20px pentru toate părțile */
    margin: 20px;

    /* Marginile de sus și de jos 10px, stânga și dreapta 20px */
    margin: 10px 20px;

    /* Marginea de sus 10px, stânga și dreapta 20px, de jos 5px */
    margin: 10px 20px 5px;

    /* Marginea de sus 10px, dreapta 15px, de jos 20px, stânga 25px */
    margin: 10px 15px 20px 25px;

    De asemenea, există proprietăți pentru setarea separată a marginii fiecărei părți.

    margin-top: 20px;
    margin-right: 10px;
    margin-bottom: 15px;
    margin-left: 30px;

    Marginea este folosită pentru a crea un spațiu între două elemente adiacente. În exemplul de mai jos, fiecare element al cardului are o bordură colorată, astfel încât se vede că între ele este o margine.

    Atentie...!!!

    Când folosiți umplutura pentru a crea goluri, elementele se vor potrivi perfect între ele fără un spațiu. Dacă elementul nu are o culoare de bordură sau de fundal, este imposibil să se determine vizual dacă marginile sunt setate corect, astfel trebuie să deschideți instrumentele de dezvoltare.

    Proprietatea padding separă conținutul de marginea elementului, în timp ce margin creează goluri între elementele adiacente. În asta constă și diferența.

    • Prăbușirea marginilor verticale

      Marginile verticale ale elementelor de bloc vecine nu se combină, ci este aleasă cea mai mare dintre ele. Restrângerea marginilor are loc doar pentru elementele de bloc în fluxul normal al documentului.

      Exemplu :

      HTML

      <div class="margins-collapse-box margins-collapse-top"></div>
      <div <div class="margins-collapse-box margins-collapse-bottom"></div>

      CSS

      .margins-collapse-box {
      width: 200px;
      height: 100px;
      }

      .margins-collapse-top {
      margin-bottom: 20px;
      background-color: teal;
      }

      .margins-collapse-bottom {
      margin-top: 40px;
      background-color: palevioletred;
      }

      Să dăm blocului de sus o margine de jos de 20px și blocului de jos o margine de sus de 40px. Din cauza prăbușirii, marginea dintre ele va fi de 40px, deoarece este aleasă cea mai mare dintre cele două.

    • Ieșirea marginilor verticale în afara elementului părinte

      Marginea verticală a unei cutii imbricate se încadrează în afara părintelui și le împinge pe ambele departe de vecin. Dacă elementul părinte are și o margine superioară, atunci va fi selectată cea mai mare dintre valori.

      Exemplu :
      HTML

      <div class="outer-box">
      <div class="inner-box"></div>
      </div˘

      CSS

      În exemplul de mai sus, am setat marginea de sus la 40px pentru div.inner-box dar, ca rezultat, nu se va îndepărta de partea de sus a div.outer-box, ci mai degrabă întreaga div .outer-box de la marginea ecranului va face acest lucru.

      .outer-box {
      width: 200px;
      height: 200px;
      background-color: teal;
      }

      .inner-box {
      width: 100px;
      height: 100px;
      margin-top: 40px;
      background-color: palevioletred;
      }

      Atentie...!!!

      Marginile de sus și de jos de la joncțiunea cu blocul părinte par să-l străpungă și să cadă, asta fiind una dintre cele mai frecvente erori de layout. Setați margini numai între două elemente adiacente.

  • Proprietatea border

    BORDER: SIZE STYLE COLOR

    O proprietate compusă care controlează lățimea, stilul și culoarea bordurii (chenar) elementului.

    border: lățime stil culoare;

    De asemenea, există proprietăți pentru setarea separată a acestor valori.

    border-width: lățimea;
    border-style: stilul;
    border-color: culoarea;

    Există multe stiluri de bordură, cele mai populare valori fiind solid, dashed și dotted.

    Exemplu :

    HTML

    <div class="border-box"></div>

    CSS

    .border-box {
    width: 100px;
    height: 100px;
    border: 5px solid tomato;
    }

    Tipuri de borduri

    • Borduri individuale

      Puteți seta stiluri individuale de bordură pentru fiecare parte a unui element. Formatul numelui proprietății este simplu – border-side-property: valoare.

      Exemplu :

      HTML

      <div class="individual-border-box"></div>

      CSS

      .individual-border-box {
      width: 100px;
      height: 100px;

      /* Setați stilurile bordurii de sus */
      border-top-width: 3px;
      border-top-style: solid;
      border-top-color: blue;

      /* Setați stilurile bordurii din dreapta */
      border-right-width: 5px;
      border-right-style: dotted;
      border-right-color: black;

      /* Setați stilurile bordurii de jos */
      border-bottom-width: 7px;
      border-bottom-style: dashed;
      border-bottom-color: palevioletred;

      /* Setați stilurile bordurii din stânga */
      border-left-width: 9px;
      border-left-style: dashed;
      border-left-color: green;
      }

    • Bordura rotunjită

      Pentru a rotunji toate colțurile bordurii există următoarea proprietate compusă border-radius. Valoarea poate fi setată în unități absolute sau relative.

      Exemplu :

      HTML

      <div class="round-border-box-1"></&div>
      <div class="round-border-box-2"></div>
      <div class="round-border-box-3"></div>

      CSS

      .round-border-box-1 {
      width: 140px;
      height: 80px;
      border-radius: 10px;
      background-color: #f44336;
      }

      .round-border-box-2 {
      width: 100px;
      height: 100px;
      border-radius: 20%;
      background-color: #2196f3;
      }

      .round-border-box-3 {
      width: 120px;
      height: 120px;
      border-radius: 50%;
      background-color: #4caf50;
      }

      Următoarele proprietăți vă permit să rotunjiți fiecare colț al bordurii în mod individual.

      /* colțul din stânga sus */
      border-top-left-radius: valoare;

      /* colțul din dreapta sus */
      border-top-right-radius: valoare;

      /* colțul din dreapta jos */
      border-bottom-right-radius: valoare;

      /* colțul din dreapta jos */
      border-bottom-left-radius: valoare;

Tipuri de elemente

În foaia de stil a browserului, fiecare element are setat tipul său – proprietatea display. Comportamentul unui element este determinat de tipul acestuia. Cele mai comune tipuri sunt elementele de tip bloc și inline. În plus, există o serie de tipuri suplimentare, cum ar fi elementele de tip inline-block.

  • Elementele de tip inline sunt concepute pentru a evidenția și proiecta conținutul textului. De exemplu, link, buton, imagine etc.
  • Elementele de tip bloc sunt folosite pentru a marca containerele de tip text (titlu, listă, paragraf) și secțiuni semantice mari (secțiune, antet, subsol).
  • Puteți afla tipul oricărui element din referință sau specificație.

Uneori este necesar ca un element de tip inline să se comporte ca un element de tip bloc și invers, astfel încât valoarea proprietății display să fie schimbată în oricare doriți.

  • Elementele de tip bloc (block-level)

    Din punct de vedere vizual, sunt zone dreptunghiulare aranjate una după alta, de sus în jos.

    • Au setată proprietatea display: block.
    • Ocupă toată lățimea disponibilă a rândului (spațiul orizontal) din interiorul părintelui.
    • Înălțimea lor este determinată de conținut, cu excepția cazului în care este setată în mod explicit în proprietatea height.
    • Încep întotdeauna din rând nou, astfel încât mai multe elemente adiacente la nivel de bloc sunt plasate vertical unul sub celălalt.
    • Pot avea orice proprietăți ale geometriei elementului: lățime, înălțime, margini, borduri și umpluturi.

    Util...!!!

    Marea majoritate a elementelor sunt de tip bloc. De exemplu (h1), (p), (section), (ul), (li), (div), (header) și multe altele.

    • Lățimea fixă

      Chiar dacă lățimea elementelor de tip bloc este setată în mod explicit, acestea vor fi totuși plasate vertical unul sub celălalt. În mod implicit, tot spațiul liber de pe o linie este ocupat de marginea dreaptă automată a elementului (margin-right).

      Exemplu :
      HTML

      <div class="fixed-width-box indigo"></div>
      <div class="fixed-width-box green"></div>
      <div class="fixed-width-box amber"></div>

      CSS

      .fixed-width-box {
      width: 240px;
      height: 80px;
      }

      .indigo {
      background-color: #3f51b5;
      }

      .green {
      background-color: #4caf50;
      }

      .amber {
      background-color: #ffc107;
      }

      Instrumente de dezvoltare

      Pentru a vedea marginea unui element, trebuie să utilizați instrumentele de dezvoltare, deoarece aceasta nu se evidențiază vizual pe pagină.

    • Centrarea orizontală a elementelor de tip bloc

      Un element de tip bloc poate fi centrat orizontal în interiorul părintelui său atunci când părintele este mai mare în lățime decât elementul respectiv. Adică, elementul de tip bloc are o lățime fixă. Pentru a face acest lucru, se folosește o caracteristică specifică a marginilor orizontale automate.

      Exemplu 1 :
      HTML

      <div class="horizontal margin-right-box indigo"></div>
      <div class="horizontal margin-right-left-box green"></div>
      <div class="horizontal margin-center-box amber"></div>

      CSS

      • În mod implicit, un element de tip bloc are o margine automată la dreapta, care ocupă tot spațiul liber de pe linie.

        .margin-right-box {
        margin-right: auto;
        }

      • Dacă setați marginea automată la stânga și la dreapta, spațiul liber este distribuit în mod egal între ele, iar elementul de tip bloc este centrat.

        .margin-right-left-box {
        background-color: tomato;
        margin-right: auto;
        margin-left: auto;
        }

      • Pentru o notare mai concisă, puteți folosi proprietatea compusă margin, setând marginile verticale la zero și cele orizontale la auto.

  • Elementele de tip inline

    Acestea sunt aranjate vizual, pe rând, pe o linie, până când este suficient spațiu pe ea, după care încep să se transfere la următoarea linie și să o umple.

    • Au setată proprietatea display: inline.
    • Lățimea și înălțimea lor depind doar de conținut, nu le puteți seta în mod explicit. Adică, proprietățile width și height nu au niciun efect.
    • Ele sunt plasate pe linie atâta timp cât există suficient spațiu pe ea, după care cele noi sunt transferate pe linia următoare.
    • Acestea pot fi setate numai cu geometrie orizontală: margini din stânga și dreapta, umplutura și bordura. Adică, se ignoră valorile de sus și de jos ale margin, padding și border.

  • Pseudo-clase suplimentare

    Aceste pseudo-clase sunt folosite în cazuri destul de rare sau foarte specifice.

    • :first-of-type – selectează primul selector de acest tip.
    • :last-of-type – selectează ultimul selector de acest tip.
    • :only-of-type – selectează un element care este singurul copil de acest tip din colecția de vecini.
    • :nth-of-type(an+b) – selectează numărul selectorului dat între paranteze, folosind formula an+b.
    • :nth-last-of-type(an+b) – analogul :nth-of-type() cu diferența că se numără de la sfârșitul colecției (al ultimului element).
    • :only-child – selectează elementul care este singurul copil cu respectivul selector din colecția vecinilor.
    • :empty – selectează elementele goale, adică fără copii și text.
